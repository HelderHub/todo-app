JavaScript Event Types

1. Mouse Events

  • click – When the user clicks an element.
  • dblclick – When the user double-clicks an element.
  • mousedown – When the mouse button is pressed.
  • mouseup – When the mouse button is released.
  • mousemove – When the mouse moves over an element.
  • mouseover – When the mouse pointer enters an element.
  • mouseout – When the mouse pointer leaves an element.
  • contextmenu – When the right mouse button is clicked.

2. Keyboard Events

  • keydown – When a key is pressed.
  • keyup – When a key is released.

3. Form Events

  • focus – When an element gains focus.
  • blur – When an element loses focus.
  • change – When the value of an input or select changes.
  • input – When a user types in an input field.
  • submit – When a form is submitted.

4. Window Events

  • load – When the page fully loads.
  • resize – When the browser window is resized.
  • scroll – When the page is scrolled.

5. Clipboard Events

  • copy – When content is copied.
  • cut – When content is cut.
  • paste – When content is pasted.

6. Drag and Drop Events

  • drag – When an element is being dragged.
  • drop – When an element is dropped.

7. Media Events

  • play – When media playback starts.
  • pause – When media playback is paused.
  • ended – When media playback ends.

8. Network Events

  • online – When the browser goes online.
  • offline – When the browser goes offline.

9. Animation and Transition Events

  • animationstart – When a CSS animation starts.
  • animationend – When a CSS animation ends.
  • transitionstart – When a CSS transition starts.
  • transitionend – When a CSS transition ends.

10. Sensor Events (Mobile)

  • deviceorientation – When the device orientation changes.
  • devicemotion – When the device detects motion.
